## 协作闭环检查

每次心跳时，执行以下检查：

```
1. 检查 memory/ 目录找 CLAUDE_CODE_DONE 标记
   ↓ 有
2. 执行任务 → 写 JARVIS_DONE

3. 同步知识库记忆（统一维基大脑）
   ↓
   读取 Z_Memory_Sync.json
   比对 version，如有更新合并到自身记忆
   重要决策/事件 → 写入 Z_Memory_Sync.json
```

## 统一维基大脑规则

**原则：Obsidian Vault 是唯一真相源**

**共享记忆文件：** `D:\桌面文件\伟力机械知识库\00_Workflow\memory\Z_Memory_Sync.json`

**读取规则：**
- 每次心跳读取 Z_Memory_Sync.json
- 比对 version，有变化则合并 entries 到自身记忆

**群消息处理规则（克劳德）：**
- 群消息 tag: `group_message`
- 1小时内：贾维斯处理，克劳德监视
- 超过1小时未解决：克劳德立刻响应，全力以赴调动所有资源应对解决
- 问题已解决：形成文档 → 上传飞书企业知识库 → 完成后删除记忆

**写入规则：**
- 重要决策、架构变更、里程碑事件 → 写入新 entry
- 格式：{id, agent:"jarvis", timestamp, content, tags, confidence}

## 快速检查

1. 读最新 memory 文件
2. 找 `CLAUDE_CODE_DONE` 标记
3. 同步知识库记忆
4. 完成任务后追加 `JARVIS_DONE` 标记
