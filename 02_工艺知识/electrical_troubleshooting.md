# 电气系统故障排除指南

## 第一章 电气系统概述

### 1.1 系统组成

工业设备的电气系统主要包括：

- **电源系统**：三相四线制 380V 电源
- **控制系统**：PLC 控制器，处理逻辑和数据
- **驱动系统**：伺服驱动器，控制电机运动
- **电机系统**：主轴电机、进给电机、辅助电机
- **传感器系统**：位置传感器、温度传感器、压力传感器
- **保护系统**：断路器、熔断器、过载保护

### 1.2 典型工作流程

1. 电源进入配电柜
2. 通过断路器分配到各个子系统
3. PLC 接收传感器信号
4. 根据程序逻辑发送控制指令
5. 驱动器控制电机运动
6. 传感器反馈实时状态

## 第二章 常见电气故障

### 2.1 无法启动

#### 症状：按下启动按钮无反应

**可能原因**：
1. 电源未接通
2. 断路器跳闸
3. 启动按钮故障
4. 控制器故障

**诊断步骤**：
1. 检查电源指示灯是否亮
2. 检查断路器状态
3. 检查启动按钮是否卡住
4. 测量控制器电源输入

**解决方案**：
- 检查电源连接
- 复位断路器
- 更换启动按钮
- 更换控制器

#### 症状：电源指示灯亮但无法启动

**可能原因**：
1. 紧急停止按钮被按下
2. 安全门开启
3. 过载保护动作
4. 控制器程序故障

**诊断步骤**：
1. 检查紧急停止按钮
2. 检查安全门开关
3. 检查过载指示灯
4. 重启控制器

**解决方案**：
- 复位紧急停止按钮
- 关闭安全门
- 等待过载保护复位
- 重新加载程序

### 2.2 主轴电机故障

#### 症状：主轴无法启动或启动后立即停止

**可能原因**：
1. 电源相序错误
2. 电机绝缘故障
3. 驱动器故障
4. 机械卡顿

**诊断步骤**：
1. 检查三相电源相序（使用相序表）
2. 测量电机绝缘阻值（应 > 2MΩ）
3. 检查驱动器指示灯
4. 手动转动主轴检查是否卡顿

**解决方案**：
- 调整电源相序
- 更换电机
- 更换驱动器
- 检查机械部分

#### 症状：主轴运行但转速不稳定

**可能原因**：
1. 驱动器参数不当
2. 编码器故障
3. 电源波动
4. 负载过大

**诊断步骤**：
1. 检查驱动器参数设置
2. 测量编码器输出信号
3. 测量电源电压波动
4. 检查主轴负载

**解决方案**：
- 调整驱动器参数
- 更换编码器
- 稳定电源供应
- 减少主轴负载

### 2.3 伺服电机故障

#### 症状：轴向移动缓慢或无法移动

**可能原因**：
1. 伺服电机故障
2. 驱动器故障
3. 编码器故障
4. 机械卡顿

**诊断步骤**：
1. 检查伺服驱动器故障指示灯
2. 测量伺服电机绝缘阻值
3. 检查编码器连接
4. 手动移动轴检查是否卡顿

**解决方案**：
- 更换伺服电机
- 更换驱动器
- 更换编码器
- 清洁导轨和丝杆

#### 症状：伺服电机发热

**可能原因**：
1. 驱动器电流设定过高
2. 电机长时间高负载运行
3. 散热不足
4. 电机故障

**诊断步骤**：
1. 测量电机温度
2. 检查驱动器电流设定
3. 检查散热风扇
4. 测量电机绝缘阻值

**解决方案**：
- 降低驱动器电流
- 减少运行时间
- 清洁散热器
- 更换电机

### 2.4 传感器故障

#### 症状：位置显示不正确

**可能原因**：
1. 位置传感器故障
2. 传感器连接松动
3. 传感器被污染
4. 控制器输入故障

**诊断步骤**：
1. 检查传感器连接
2. 清洁传感器
3. 测量传感器输出信号
4. 检查控制器输入

**解决方案**：
- 紧固传感器连接
- 清洁传感器
- 更换传感器
- 更换控制器输入模块

#### 症状：温度显示异常

**可能原因**：
1. 温度传感器故障
2. 传感器接线错误
3. 显示模块故障
4. 传感器被污染

**诊断步骤**：
1. 用标准温度计对比测量
2. 检查传感器接线
3. 测量传感器输出信号
4. 检查显示模块

**解决方案**：
- 更换温度传感器
- 纠正接线
- 更换显示模块
- 清洁传感器

## 第三章 电源系统维护

### 3.1 电源检查

**每日检查**：
- 检查电源指示灯
- 观察电压表读数
- 检查是否有异常声音

**每周检查**：
- 测量三相电压（应为 380V ±10%）
- 测量三相电流平衡度（应 < 10%）
- 检查断路器是否有过热迹象

**每月检查**：
- 清洁配电柜内部
- 检查所有接线端子
- 测量接地电阻（应 < 4Ω）

### 3.2 电源故障排除

| 故障现象 | 可能原因 | 解决方案 |
|--------|--------|--------|
| 电压过低 | 电源不稳定或负载过大 | 检查电源、减少负载 |
| 电压过高 | 稳压器故障 | 更换稳压器 |
| 三相不平衡 | 单相负载过大 | 重新分配负载 |
| 频繁跳闸 | 短路或过载 | 检查电路、减少负载 |

## 第四章 控制系统维护

### 4.1 PLC 控制器维护

**定期检查**：
- 检查控制器指示灯状态
- 检查冷却风扇运行
- 检查内存使用情况
- 备份程序文件

**故障排除**：

| 指示灯 | 状态 | 含义 | 处理方法 |
|------|------|------|--------|
| 电源灯 | 不亮 | 无电源 | 检查电源连接 |
| 运行灯 | 闪烁 | 程序运行 | 正常 |
| 故障灯 | 亮 | 系统故障 | 查看故障代码 |
| 通信灯 | 闪烁 | 通信正常 | 正常 |

### 4.2 程序备份与恢复

**备份步骤**：
1. 连接编程工具到控制器
2. 读取当前程序
3. 保存到计算机
4. 定期更新备份

**恢复步骤**：
1. 连接编程工具到控制器
2. 打开备份程序文件
3. 下载到控制器
4. 验证程序正确性

## 第五章 电气安全规范

### 5.1 安全注意事项

1. **高压危险**：
   - 380V 电源可能致命
   - 禁止在带电情况下工作
   - 必须切断电源才能维修

2. **触电防护**：
   - 穿绝缘鞋
   - 使用绝缘工具
   - 戴绝缘手套
   - 不在潮湿环境工作

3. **火灾防护**：
   - 配电柜周围不堆放易燃物
   - 定期检查电气连接
   - 及时更换老化的电缆

### 5.2 应急处理

**触电应急**：
1. 立即切断电源
2. 将伤者移离电源
3. 进行人工呼吸或心肺复苏
4. 立即拨打急救电话

**火灾应急**：
1. 立即切断电源
2. 使用干粉灭火器灭火
3. 禁止用水灭火
4. 疏散人员

## 第六章 故障代码参考

| 代码 | 含义 | 原因 | 解决方案 |
|------|------|------|--------|
| E001 | 电源故障 | 电源不稳定 | 检查电源 |
| E002 | 电机故障 | 电机过热或故障 | 检查电机 |
| E003 | 传感器故障 | 传感器故障或脱落 | 检查传感器 |
| E004 | 驱动器故障 | 驱动器过热或故障 | 检查驱动器 |
| E005 | 通信故障 | 通信中断 | 检查连接 |

## 第七章 定期维护计划

### 7.1 日常维护（每天）
- 检查电源和指示灯
- 观察控制器运行状态
- 检查是否有异常声音或气味

### 7.2 周期维护（每周）
- 测量电源电压
- 检查断路器状态
- 清洁配电柜外部

### 7.3 定期维护（每月）
- 全面检查电气系统
- 测试紧急停止功能
- 检查接地电阻

### 7.4 大修保养（每年）
- 更新控制器程序
- 更换老化的电缆
- 全面检测电气安全
