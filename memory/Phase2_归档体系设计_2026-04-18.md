## 二、故障记录单模板

每条故障生成一个独立 Markdown 文件：

```markdown
# 故障处理记录单

## 基本信息
- 记录编号：TRB-2026-04-18-001
- 客户名称：张先生
- 设备型号：WLA-200L
- 设备编号：WL20240315001
- 故障类型：机械类 > 加热圈故障
- 故障代码：E01
- 紧急程度：⚠️ 紧急
- 报告时间：2026-04-18 09:00
- 处理状态：✅ 已解决

## 故障描述
[客户原始描述]

## AI初步分析
- 可能原因①：加热圈损坏
- 可能原因②：温度传感器故障
- 可能原因③：原料结块

## 处理过程
| 时间 | 操作人 | 操作内容 | 结果 |
|------|--------|----------|------|
| 09:00 | 张先生 | 报告故障 | - |
| 09:15 | 技术李工 | 检查加热圈 | 发现2号加热圈损坏 |
| 09:30 | 技术李工 | 更换加热圈 | 恢复正常 |

## 解决方案
加热圈损坏，已更换同规格加热圈。

## 根本原因
加热圈使用年限超过3年，老化导致接触不良。

## 预防建议
建议对使用超过2年的加热圈进行预防性更换。

## 归档信息
- 归档时间：2026-04-18 10:30
- 归档目录：2026/01_连续式挤吹/WLA-200L/报警代码类/E01/
- FAQ更新：✅ 已更新 → FAQ/05_故障处理.md
- 相关附件：trb-001-photo-01.jpg, trb-001-video-01.mp4
```

**文件名格式：** `TRB-{年}-{月}-{日}-{序号}.md`
**示例：** `TRB-2026-04-18-001.md`

## 三、故障类型分类体系

### 一级分类（6类）
1. **机械类** — 螺杆、轴承、传动、模头
2. **电气类** — 报警代码、电机、传感器、控制器
3. **液压类** — 油泵、阀门、液压缸、漏油
4. **温度/加热类** — 加热圈、温度传感器、冷却系统
5. **产品质量类** — 不出料、壁厚不均、黑点黄线
6. **其他** — 操作失误、原料问题、配件损坏

### 二级分类（报警代码）
```
E00 - 系统正常
E01 - 加热圈故障
E02 - 温度传感器故障
E03 - 电机过载
E04 - 油压异常
E05 - 紧急停止
E06 - 门未关闭
E07 - 气压异常
E08 - 原料短缺
E09 - 模头温度异常
E10 - 壁厚控制系统故障
...
EXX - 待补充
```
