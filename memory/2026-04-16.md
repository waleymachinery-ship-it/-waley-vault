## Watchdog优化完成（13:58）

### 问题
Hermes Gateway 循环重启 - Watchdog 旧逻辑缺陷
- 旧逻辑：看到 error_code 就重启，不看 gateway_state 是否running
- 导致：即使 Hermes 正常，Watchdog 仍不断重启

### 解决方案
优化 hermes-gateway-watchdog.ps1（v2）：
1. 消除循环重启：只看进程是否存在 + gateway_state 是否running
2. 增加5分钟冷却期：重启后5分钟内不再重启
3. 错误状态自动清除：重启前清除旧 error_code
4. 多次验证机制：重启后验证3次，确认成功
