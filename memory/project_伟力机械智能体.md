## 当前系统架构

```
用户消息 → OpenClaw（微信入口）
              ↓
         写入 Vault / memory
              ↓
         AI推理判断 → 执行/回复
              ↓
         回复 / 更新知识库
              ↓
         Hermes检测CLAUDE_CODE_DONE → 执行 → 写HERMES_DONE
```
