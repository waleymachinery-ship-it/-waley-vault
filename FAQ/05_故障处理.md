# FAQ 05：故障处理

> 故障处理 | 更新：2026-04-10

## 常见故障

### 不出料 / 出料慢

**可能原因**：
1. 螺杆磨损或损坏
2. 温度设定过低
3. 原料里有杂质堵塞
4. 换网器堵塞

**排查步骤**：
1. 检查温度是否达到设定值
2. 清理换网器和口模
3. 如仍不出料，联系维修

---

### 产品壁厚不均

**可能原因**：
1. 壁厚程序设置不当
2. 模头间隙不均匀
3. 吹胀比不稳定
4. 材料流动性变化

**排查步骤**：
1. 检查壁厚程序参数
2. 检测模头口模间隙
3. 检查气压是否稳定

---

### 产品表面有黑点/黄线

**可能原因**：
1. 材料过热降解
2. 螺杆或料筒磨损
3. 换色不彻底
4. 原料受污染

**排查步骤**：
1. 降低加工温度
2. 进行彻底洗料/换色
3. 检查螺杆和料筒磨损情况
4. 更换原料

---

### 声音异常/振动大

**可能原因**：
1. 轴承磨损
2. 润滑不足
3. 螺杆弯曲
4. 地脚螺栓松动

**处理**：立即停机检查，联系维修
